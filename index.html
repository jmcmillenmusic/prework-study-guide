HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use semantic elements when possible (for example, "section" instead of "div").

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all files to stage your changes
  • git commit -m "Making changes": commits your changes with a commit message inside quotes
  • git pull origin main: pulls the main branch to make sure you have the most updated stuff before you push
  • git push origin branch-name: pushes your changes from your branch (branch-name) to main
  • Use git checkout main and git pull origin main to update locally after you push your changes!

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions must be declared in your code before you call them later.
  • For loops have 3 conditional statements: the starting point, the condition that breaks out of the loop, and the incrementation of the loop.
